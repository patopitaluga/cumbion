# Cumbion

Dynamically typed Turing-complete programming language that's also cumbia lyrics.

Live javascript JavaScript parser and interpreter: https://patopitaluga.github.io/cumbion/

## 'Hello, World' in Cumbion
```
¡Y dice! "Hello World"
```
